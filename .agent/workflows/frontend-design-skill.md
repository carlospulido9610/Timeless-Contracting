---
description: Guía para la creación de interfaces frontend distintivas, de grado de producción, evitando estéticas genéricas de IA.
---

Esta skill guía la creación de interfaces frontend excepcionales con atención a los detalles estéticos y opciones creativas audaces.

### Pensamiento de Diseño
Antes de codificar, asume una dirección estética clara (mínima, maximalista, retro-futurista, lujo, editorial, etc.):
1. **Propósito**: ¿Qué problema resuelve?
2. **Tono**: Elige un extremo estético y ejecútalo con precisión.
3. **Diferenciación**: ¿Qué hará que esta interfaz sea INOLVIDABLE?

### Directrices Estéticas
*   **Tipografía**: Usa fuentes hermosas y únicas. Evita Arial e Inter. Empareja una fuente de display distintiva con una de cuerpo refinada.
*   **Color y Tema**: Comprométete con una estética cohesiva. Usa variables CSS.
*   **Movimiento**: Usa animaciones para micro-interacciones. Prioriza CSS o la librería Framer Motion.
*   **Composición Espacial**: Diseños inesperados, asimetría, flujos diagonales, espacios negativos generosos.
*   **Detalles Visuales**: Crea atmósfera con texturas de ruido, mallas de degradado, transparencias en capas y sombras dramáticas.

### Lo que NO se debe hacer
- **NUNCA** uses estéticas genéricas generadas por IA (Inter, esquemas de color trillados como degradados púrpuras sobre blanco, layouts predecibles).
- **NUNCA** converjas en opciones comunes. Cada diseño debe ser único.

### Implementación
Ejecuta el código (HTML/CSS/JS, React, etc.) asegurando que sea:
- Grado de producción y funcional.
- Visualmente impactante y memorable.
- Cohesivo con un punto de vista estético claro.
- Meticulosamente refinado en cada detalle.
